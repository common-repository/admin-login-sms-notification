== Description ==

Admin login SMS notification
Sends SMS notification when admin logged in blog (for sending SMS using API from service SMS.RU).

To get api key you need to register on site sms.ru

== Installation ==

1. Upload `wp-admin-login-sms-notification` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the "Settings | SMS Login Notification" administration page to enter api-key, phone number and enable notifications.
